# Моделирование лесных пожаров (Клеточный автомат)

## **Состояние клетки:**
$$S_t(x,y) \in \{\text{Empty}, \text{Tree}, \text{Burning}, \text{Water}, \text{Charred}\}$$

**Окрестность:** Мура (8 соседей).
**Время:** Дискретное $t = 0, 1, 2...$

---

## Базовые правила 

1. **Рост дерева:**
   $$P(S_{t+1}=\text{Tree} \mid S_t=\text{Empty}) = p_g$$

2. **Молния (спонтанное возгорание):**
   $$P(S_{t+1}=\text{Burning} \mid S_t=\text{Tree}) = p_l$$

3. **Распространение от соседа:**
   $$P_{\text{base}} = p_b$$

---

## 3. Дополнительные правила

### 3.1. Влияние ветра на распространение
Вероятность воспламенения зависит от угла между ветром и направлением на соседа:
$$P_{\text{ign}} = \min(1, p_b + s_w \cdot \cos\theta)$$
$$\cos\theta = \frac{(-dx)w_x + (-dy)w_y}{\sqrt{dx^2 + dy^2}}$$
где $\vec{w} = (\cos\alpha, -\sin\alpha)$ — единичный вектор направления ветра.

### 3.2. Динамика направления ветра
Угол ветра изменяется случайным образом каждый такт:
$$\alpha_{t+1} = (\alpha_t + \Delta\alpha) \mod 360^\circ$$
$$\Delta\alpha \sim \mathcal{U}[-6^\circ, +6^\circ]$$

### 3.3. Система дождевых облаков (Спавн)
Облако создается с наветренной стороны, если их количество меньше лимита:
$$P(\text{spawn}) = p_c, \quad \text{если } |\mathcal{C}| < C_{\max}$$
Координаты спавна зависят от знака компонент вектора $\vec{w}$. (от направления ветра)

### 3.4. Движение облаков
Облака перемещаются по вектору ветра:
$$\vec{pos}_{t+1} = \vec{pos}_t + \vec{w} \cdot s_w \cdot v_{\text{move}}$$
Удаляются при выходе за границы экрана ($D_{\text{out}}$).

### 3.5. Дождь (Тушение огня)
Клетка тушится мгновенно, если попадает в радиус любого облака:
$$R_t(x,y) = \bigvee_{c \in \mathcal{C}_t} \left[ (x-x_c)^2 + (y-y_c)^2 < r_c^2 \right]$$
$$\text{Если } R_t(x,y) = \text{true} \implies S_{t+1} = \text{Charred}$$

### 3.6. Ограниченное топливо (Время горения)
Клетка горит конечное число тактов ($f_{\max}=3$):
$$f_{t+1} = f_t - 1$$
$$\text{Если } f_t \leq 1 \implies S_{t+1} = \text{Charred}$$

### 3.7. Состояние «Пепел» (Charred)
Задержка перед восстановлением земли:
$$\text{Burning} \to \text{Charred} \xrightarrow{1 \text{ такт}} \text{Empty}$$

### 3.8. Водные препятствия
Клетки воды неизменны и блокируют огонь:
$$S_t(x,y) = \text{Water} \implies S_{t+1}(x,y) = \text{Water}$$

## 4. Параметры симуляции

| Параметр | Обозначение | Значение |
|----------|-------------|----------|
| Вероятность роста | $p_g$ | 0.005 |
| Вероятность молнии | $p_l$ | 0.001 |
| База распространения | $p_b$ | 0.1 |
| Сила ветра | $s_w$ | [0, 1] |
| Макс. облаков | $C_{\max}$ | 5 |
| Топливо | $f_{\max}$ | 3 |
| Скорость облака | $v_{\text{move}}$ | 5.0 (клеток при силе ветра 1.0) |

### 4.1. Обозначения символов в формулах

| Символ | Описание | Значение / Диапазон |
|--------|----------|---------------------|
| $dx, dy$ | Смещение координат соседа | $\{-1, 0, 1\}$ |
| $w_x, w_y$ | Компоненты вектора ветра | $[-1, 1]$ |
| $\alpha$ | Угол направления ветра | $[0^\circ, 360^\circ)$ |
| $r_c$ | Радиус дождевого облака | $[22, 55]$ |
| $D_{\text{out}}$ | Дистанция удаления облака | 150 |
| $\mathcal{C}$ | Множество активных облаков | — |
| $\mathcal{U}$ | Равномерное распределение | — |
