## Задание

1. Реализовать алгоритм проведения серии экспериментов по генерации дискретной случайной величины (ДСВ), заданной рядом распределения

3. Вычислить статистику хи-квадрат и применить критерий хи-квадрат при разных объёмах выборки $N$ ($N = 10, 100, 1000, 10000$)
4. Сделать вывод

## Теоретические сведения

### Случайные объекты и базовый датчик

Случайные объекты:
1. Случайные события
2. **Случайные величины**
3. Случайные процессы
4. Случайные потоки событий

Базовый датчик (БД) генерирует $\alpha \in [0; 1]$. Через преобразование $\alpha$ получаем нужный случайный объект $X$.

### Дискретная случайная величина (ДСВ)

ДСВ задаётся **рядом распределения** — таблицей значений и их вероятностей:

| $x$   | $x_1$ | $x_2$ | $\ldots$ | $x_m$ |
|-------|--------|--------|----------|--------|
| $p$   | $p_1$  | $p_2$  | $\ldots$ | $p_m$  |

$$p_i = P\{x = x_i\}, \quad \sum_{i=1}^{m} p_i = 1$$

Каждому значению $x_i$ соответствует событие $A_i$ = «случайная величина приняла значение $x_i$». Эти события образуют полную группу попарно несовместных событий.

### Алгоритм генерации ДСВ

Тот же алгоритм, что и для генерации события из полной группы, но с присвоением значения $x_k$:

1. $A := \alpha$ (из базового датчика), $k := 1$
2. $A := A - p_k$
3. Если $A \leq 0$, то $x := x_k$; иначе $k := k + 1$, перейти к шагу 2

### Специальные методы генерации ДСВ

#### Равномерное дискретное распределение

От 0 до $n$: $p_i = \frac{1}{n+1}$, датчик: $x = \text{Int}(\alpha \cdot (n+1))$

От $a$ до $b$: $n = b - a$, $x = \text{Int}(\alpha \cdot (n+1)) + a$

Для $x$ из $\{1, 2, \ldots, n\}$: $x = \text{Int}(\alpha \cdot n) + 1$

#### Геометрическое распределение

Число неудач до первого успеха. $P\{x = m\} = p(1-p)^m$

Датчик: $x = \text{Int}\left(\frac{\ln\alpha}{\ln(1-p)}\right)$

#### Отрицательное биномиальное распределение

Число неудач до $r$-го успеха. $P\{x = m\} = C_{m+r-1}^{m} \cdot p^r (1-p)^m$

Датчик: $x = \sum_{i=1}^{r} \text{Int}\left(\frac{\ln\alpha_i}{\ln(1-p)}\right)$

#### Биномиальное распределение

Число успехов в $n$ испытаниях. $P\{x = m\} = C_n^m p^m (1-p)^{n-m}$

Датчик: $x = \sum_{k=1}^{n} \theta(p - \alpha_k)$, где $\theta(x) = 1$ при $x \geq 0$, иначе $0$ (ступенчатая функция Хевисайда)

#### Распределение Пуассона

Число событий с интенсивностью $\lambda$. $P\{x = m\} = \frac{\lambda^m}{m!} e^{-\lambda}$

Датчик: $x = \min\left\{m : \sum_{i=0}^{m} \ln\alpha_i < -\lambda\right\}$

Алгоритм: $m := 0$, $S := 0$; цикл: $\alpha$ из БД, $S := S + \ln\alpha$; если $S < -\lambda$, то $x = m$, иначе $m := m + 1$.

### Статистическая обработка результатов для ДСВ

**Эмпирическая вероятность (относительная частота):**
$$\hat{p}_i = \frac{n_i}{N}$$

где $n_i$ — частота значения $x_i$ (сколько раз наблюдалось это значение), $N$ — объём выборки.

**Математическое ожидание (теоретическое):**
$$E\,x = \sum_{i=1}^{m} p_i \cdot x_i$$

**Дисперсия (теоретическая):**
$$D\,x = \sum_{i=1}^{m} p_i (x_i - E\,x)^2 = \sum_{i=1}^{m} p_i \cdot x_i^2 - (E\,x)^2$$

**Эмпирическое среднее:**
$$\hat{E}\,x = \sum_{i=1}^{m} \hat{p}_i \cdot x_i$$

**Эмпирическая дисперсия:**
$$\hat{D}\,x = \sum_{i=1}^{m} \hat{p}_i \cdot x_i^2 - (\hat{E}\,x)^2$$

**Абсолютные погрешности:**
$$\Delta_E = |\hat{E}\,x - E\,x|, \quad \Delta_D = |\hat{D}\,x - D\,x|$$

**Относительные погрешности:**
$$\delta_E = \frac{\Delta_E}{|E\,x|}, \quad \delta_D = \frac{\Delta_D}{|D\,x|}$$

### Критерий хи-квадрат

Статистика хи-квадрат:

$$\hat{X}_N^2 = \sum_{i=1}^{m} \frac{n_i^2}{N \cdot p_i} - N$$

Гипотеза $H_0$ (эмпирическое распределение соответствует теоретическому) **отвергается** тогда и только тогда, когда:

$$\hat{X}_N^2 > \chi^2_{1-\alpha,\, m-1}$$

где $\alpha$ — уровень значимости.

При $\alpha = 0.05$ и $m - 1 = 4$ степенях свободы: $\chi^2_{0.95,\,4} = 9.488$.

### Критерий хи-квадрат для ДСВ с неограниченным числом значений

Для распределений с бесконечным числом значений (геометрическое, Пуассона и т.д.) нужно найти $X$ такое, что:

$$\sum_{i=X}^{\infty} p_i \leq \min_{i < X} p_i$$

Все значения $\geq X$ объединяются в одну группу «$> X$» с суммарной вероятностью.

## Вывод

Реализован алгоритм генерации дискретной случайной величины по заданному ряду распределения. Алгоритм основан на разбиении отрезка $[0; 1)$ на интервалы длиной $p_i$ и определении, в какой интервал попало случайное число из базового датчика (последовательное вычитание вероятностей).

По результатам экспериментов:

- При малых объёмах выборки ($N = 10$) эмпирические характеристики значительно отклоняются от теоретических, относительные погрешности $\delta_E$ и $\delta_D$ велики.
- С ростом $N$ выборочные среднее и дисперсия сходятся к теоретическим значениям. При $N = 10\,000$ относительные погрешности обычно составляют единицы процентов.
- Критерий хи-квадрат: при малых $N$ ($N = 10$) гипотеза $H_0$ может отвергаться из-за больших флуктуаций. При $N \geq 1000$ гипотеза о соответствии эмпирического распределения теоретическому, как правило, принимается ($\hat{X}^2_N < 9.488$).
- Это согласуется с законом больших чисел: при увеличении объёма выборки относительные частоты $\hat{p}_i$ стремятся к теоретическим вероятностям $p_i$.
