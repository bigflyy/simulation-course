## 2. Краткие теоретические сведения

### 2.1. Непрерывный марковский процесс

Марковский процесс с дискретными состояниями и непрерывным временем характеризуется **матрицей интенсивностей (генератором) Q**:

```
        | q₁₁  q₁₂  q₁₃ |
    Q = | q₂₁  q₂₂  q₂₃ |
        | q₃₁  q₃₂  q₃₃ |
```

где:
- `qᵢⱼ = λᵢⱼ` — интенсивность перехода из состояния i в состояние j (при i ≠ j)
- `qᵢᵢ = -Σ(qᵢⱼ, j≠i)` — диагональный элемент (отрицательная сумма исходящих интенсивностей)
- Каждая строка матрицы Q суммируется в 0

### 2.2. Алгоритм моделирования

**Шаг 1.** Находясь в состоянии i, генерируем время до следующего перехода по экспоненциальному закону:

```
Δt = -ln(U) / λᵢ,  где U ~ Uniform(0, 1),  λᵢ = -qᵢᵢ
```

**Шаг 2.** Выбираем следующее состояние j с вероятностью:

```
P(переход в j | из i) = qᵢⱼ / λᵢ,  j ≠ i
```

**Шаг 3.** Накопление статистики — подсчёт общего времени, проведённого в каждом состоянии.

**Шаг 4.** Эмпирические вероятности состояний:

```
πᵢ(эмп) = Tᵢ / T_общ
```

где Tᵢ — суммарное время в состоянии i, T_общ — общее время моделирования.

### 2.3. Стационарное распределение

Теоретическое стационарное распределение π находится из системы уравнений:

```
π · Q = 0
Σ(πᵢ) = 1
```

Решается методом Гаусса. Сравнение эмпирических и теоретических вероятностей позволяет оценить качество моделирования.

### 5.2. Выбор следующего состояния

```csharp
double r = rand.NextDouble();
double cumulative = 0;
for (int j = 0; j < 3; j++)
{
    if (j == i) continue;
    double transProb = Q[i, j] / lambda[i];
    cumulative += transProb;
    if (r < cumulative)
    {
        nextState = j + 1;
        break;
    }
}
```

### 5.3. Расчёт стационарного распределения (метод Гаусса)

```csharp
// Система: pi * Q = 0, sum(pi) = 1
// Транспонируем Q, заменяем последнюю строку на (1,1,1)
double[,] A = new double[3, 3];
for (int i = 0; i < 3; i++)
    for (int j = 0; j < 3; j++)
        A[j, i] = Q[i, j];
A[2, 0] = 1; A[2, 1] = 1; A[2, 2] = 1;
double[] b = { 0, 0, 1 };
```

## 8. Контрольные вопросы

**1. Что такое марковский процесс?**
— Стохастический процесс, у которого будущее состояние зависит только от текущего состояния и не зависит от предыдущих (свойство Маркова / отсутствие памяти).

**2. Что такое матрица интенсивностей Q?**
— Матрица, элементы которой qᵢⱼ задают интенсивность перехода из состояния i в j. Диагональные элементы qᵢᵢ = -Σ(qᵢⱼ, j≠i). Строки суммируются в 0.

**3. Как моделируется время до перехода?**
— По экспоненциальному распределению: Δt = -ln(U) / λᵢ, где λᵢ = -qᵢᵢ — суммарная интенсивность выхода из состояния i.

**4. Как определяется следующее состояние?**
— Вероятность перехода i→j равна qᵢⱼ / λᵢ (для j ≠ i). Выбор осуществляется по методу обратной функции распределения.

**5. Что такое стационарное распределение?**
— Вектор вероятностей π, удовлетворяющий π·Q = 0 при Σπᵢ = 1. Описывает долгосрочные доли времени, проводимые процессом в каждом состоянии.

**6. Почему эмпирические вероятности приближаются к теоретическим?**
— В силу закона больших чисел: при увеличении объёма выборки средние эмпирические значения сходятся к математическим ожиданиям.
