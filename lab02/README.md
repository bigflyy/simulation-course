### Метод конечных разностей для уравнения теплопроводности

**Задание:**
Реализовать моделирование изменения температуры в пластине на основе одномерного уравнения теплопроводности с использованием метода конечных разностей.

Выполнить моделирование с различными шагами по времени и по пространству.
Заполнить таблицу значений температуры в центральной точке пластины после 2 секунд модельного времени.

**Параметры:**
- Материал: медь (Rho = 8900 кг/м3, Cp = 385 Дж/(кг*C), Lambda = 401 Вт/(м*C))
- Толщина пластины: L = 0.1 м
- Начальная температура: T0 = 0 C
- Температура слева: Tleft = -200 C
- Температура справа: Tright = 20 C

**Результаты (температура в центре пластины после 2 секунд):**

| dt \ dx | 0.1 | 0.01 | 0.001 | 0.0001 | 0.00005 |
|---------|---------|---------|---------|---------|---------|
| 0.1 | 20.0000 | -13.6349 | -4.6126 | -4.1189 | -4.0931 |
| 0.01 | 20.0000 | -13.5461 | -4.3312 | -3.8351 | -3.8093 |
| 0.001 | 20.0000 | -13.5372 | -4.3015 | -3.8051 | -3.7793 |
| 0.0001 | 20.0000 | -13.5363 | -4.2985 | -3.8020 | -3.7762 |
| 0.00005 | 20.0000 | -13.5363 | -4.2984 | -3.8019 | -3.7761 |

**Вывод:**

Уравнение теплопроводности — это дифференциальное уравнение, точное аналитическое решение которого в общем случае сложно получить. Метод конечных разностей (метод сеток) заменяет непрерывную задачу дискретной: пластина разбивается на конечное число точек с шагом dx, а время — на шаги dt. Производные заменяются конечными разностями:
- Производная по времени: dT/dt -> (T_new - T_old) / dt
- Вторая производная по пространству: d2T/dx2 -> (T[i-1] - 2*T[i] + T[i+1]) / dx^2

Был выбран медный материал: при толщине пластины 0.1 м и времени наблюдения 2 секунды сталь практически не успевает прогреться (температура в центре остаётся близкой к начальной 0 C), тогда как в меди за это же время температура в центре уже достигает -3.8 C, что позволяет наблюдать динамику процесса. 

Так как решение приблизительное, метод сходится при уменьшении шагов по пространству и по времени. При dx = 0.1 сетка слишком грубая (всего 2 точки) и результат не отражает реальный процесс (температура остаётся 20 C). При dx = 0.01 результат уже ближе к реальности, но отличается от мелких сеток. Начиная с dx = 0.0001 и dt = 0.0001 результат практически стабилизируется на значении около -3.80 C.
