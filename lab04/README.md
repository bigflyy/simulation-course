# Лабораторная работа 4. Базовый датчик случайных чисел

## Задание

1. Реализовать базовый датчик (мультипликативный конгруэнтный генератор)
2. Вычислить выборочные среднее и дисперсию для реализованного датчика и для встроенного в язык программирования (размер выборки 100 тыс. значений)
3. Сравнить их с теоретическими
4. Сделать вывод

## Теоретические сведения

### Базовый датчик

Базовый датчик (БД) — алгоритм (компонент ПО или устройство), который генерирует реализации $\alpha$ непрерывной случайной величины, **равномерно распределённой в интервале $[0; 1)$**.

Схема работы: БД генерирует $\alpha \in [0;1]$, затем через преобразование получаем нужную случайную величину $X$.

### Типы базовых датчиков

1. ~~Физические~~
2. ~~Табличные (таблицы Брадиса)~~
3. **Псевдослучайные** — используются на практике

### Псевдослучайный датчик

Рекуррентная формула:

$$x_{n+1} = f(x_n)$$

или

$$x_{n+1} = f(x_n, x_{n-1}, \ldots)$$

$x_0$ — **зерно (seed)**, начальное значение, задаётся пользователем.

Зерно в:
- **C#**: `Random rnd = new Random()` или `Random rnd = new Random(x)`, затем `rnd.NextDouble()`

### Требования к базовым датчикам

**Математические:**
- Равномерное распределение
- Отсутствие корреляции
- Апериодичность
- Воспроизводимость

**Вычислительные:**
- Быстрота

**Общие:**
- Портируемость

### Мультипликативный конгруэнтный генератор

Формула рекуррентного вычисления:

$$x_i^* = (\beta \cdot x_{i-1}^*) \mod M$$

$$x_i = \frac{x_i^*}{M}, \quad i = 1, 2, \ldots$$

Параметры:
- $M = 2^{63} = 9\,223\,372\,036\,854\,775\,808$ — модуль
- $\beta = 2^{32} + 3 = 4\,294\,967\,299$ — множитель
- $x_0^* = \beta = 4\,294\,967\,299$ — зерно (начальное значение)

Другие методы генерации:
- Метод середины квадрата
- Метод Фиббоначи
- Регистровые сдвиги (в том числе «вихрь Мерсена»)

### Теоретические характеристики $U(0, 1)$

Для непрерывной равномерной величины на $[0; 1]$:

$$E[x] = 0.5$$

$$D[x] = \frac{1}{12} \approx 0.0833$$

### Выборочные оценки

Выборочное среднее:

$$\bar{x} = \frac{1}{N}\sum_{i=1}^{N} x_i$$

Несмещённая выборочная дисперсия:

$$S^2 = \frac{1}{N-1}\sum_{i=1}^{N}(x_i - \bar{x})^2 = \frac{1}{N-1}\left(\sum_{i=1}^{N} x_i^2 - N\bar{x}^2\right)$$

## Вывод

Реализован мультипликативный конгруэнтный генератор псевдослучайных чисел с параметрами $M = 2^{63}$, $\beta = 2^{32} + 3$. Метод использует всего два простейших алгебраических действия (умножение и взятие остатка). Проведено сравнение с встроенным генератором `System.Random` (использует вихрь Мерсена) на выборке из 100 000 значений.

Оба датчика показывают выборочные характеристики (среднее и дисперсию), близкие к теоретическим значениям равномерного распределения $U(0, 1)$: $E = 0.5$, $D = 1/12 \approx 0.0833$. Абсолютные погрешности обоих генераторов одного порядка ($\sim 10^{-4}$), что подтверждает пригодность реализованного датчика для задач стохастического моделирования.

Мультипликативный конгруэнтный метод, несмотря на свою простоту, показывает высокую точность, сопоставимую со сложными алгоритмами (вихрь Мерсена), встроенными в классические языки программирования.

С увеличением объёма выборки выборочные характеристики сходятся к теоретическим, что согласуется с законом больших чисел.
