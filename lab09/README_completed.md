## Теория

### Система M/M/1/0

Обозначение Кэндалла **M/M/1/0** означает:

- **M** (Markovian) — экспоненциальное распределение времени между прибытиями
- **M** (Markovian) — экспоненциальное распределение времени обслуживания
- **1** — один сервер
- **0** — ноль мест ожидания (очереди нет)

Пришедшая заявка обслуживается, если сервер свободен. Если сервер занят — заявка **отказывается** (теряется).

### Предельные вероятности

Граф переходов состоит из двух состояний:

```
    λ
E₀ ⟷ E₁
    μ
```

где:
- **E₀** — сервер свободен
- **E₁** — сервер занят

Уравнения Чепмена — Колмогорова в стационарном режиме:

$$-\lambda P_0 + \mu P_1 = 0$$
$$P_0 + P_1 = 1$$

Решение:

$$P_0 = \frac{1}{1 + \rho}, \qquad P_1 = \frac{\rho}{1 + \rho}$$

где $\rho = \lambda/\mu$ — коэффициент загрузки.

### Вероятность отказа

Для системы M/M/1/0 вероятность отказа совпадает с вероятностью занятости сервера (формула Эрланга B при $n = 1$):

$$P_{\text{отк}} = P_1 = \frac{\rho}{1 + \rho}$$

### Пропускная способность

Среднее число обслуженных заявок в единицу времени:

$$A = \lambda \cdot P_0 = \frac{\lambda}{1 + \rho}$$

### Среднее число занятых приборов

$$\bar{N} = 0 \cdot P_0 + 1 \cdot P_1 = \frac{\rho}{1 + \rho}$$

### Зависимость P_отк от ρ

| $\rho$ | $P_0$ | $P_1 = P_{\text{отк}}$ |
|--------|-------|------------------------|
| 0.25 | 0.8000 | 0.2000 |
| 0.50 | 0.6667 | 0.3333 |
| 0.80 | 0.5556 | 0.4444 |
| 1.00 | 0.5000 | 0.5000 |
| 2.00 | 0.3333 | 0.6667 |
| 4.00 | 0.2000 | 0.8000 |

При $\rho \to \infty$ почти все заявки получают отказ. При $\rho \to 0$ сервер почти всегда свободен.

## Как работает программа

**Симуляция N независимых заявок:**

1. Генерируются времена прибытия N заявок с интервалами ~ Exp(λ)
2. Если сервер свободен — заявка принята, время обслуживания ~ Exp(μ)
3. Если сервер занят — заявка отказана (счётчик отказов)
4. Подсчитываются доли принятых и отказанных заявок

**Результаты:**
- Столбчатая диаграмма: эмпирические vs теоретические P₀ и P₁
- Вероятность отказа, пропускная способность, число отказов
- Автоматический вывод о согласовании с теорией

## Выводы

Ожидается:
1. Эмпирическая вероятность отказа близка к теоретической $P_{\text{отк}} = \rho/(1+\rho)$
2. Пропускная способность близка к $\lambda \cdot P_0$
3. При увеличении N расхождение уменьшается
4. При $\rho > 1$ более половины заявок получают отказ
